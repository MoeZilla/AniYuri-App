## Features

- Recent Anime Release.
- Popular Anime.
- Search Anime. 
- WhatAnime (photo search anime)
- Anime Schedule.
- Top Airing Anime.
- Anime Movies.
- Download Anime.
- Automatic Download Anime.
- Dark & Light Moade
- Add to favorites.
- Sign into My Anime List.
- Auto Quality for Video Playback.
- Directly skip to next/previous episode from player.
- Update MAL list.
- FanDub Anime
- Ad free
